    Projetos Recentes

    Machine Learning:

    • Prevendo salários: Desenvolvimento de um modelo de machine learning para prever o salário de pessoas que trabalham na área de dados, usando features como: gênero, nível educacional, idade, título do cargo que ocupam e anos de experiência.
    • Estudo de SGDRegressor: O SGDRegressor é um modelo de regressão linear baseado no Gradiente Descendente Estocástico (SGD, do inglês Stochastic Gradient Descent), que é um método de otimização para encontrar os coeficientes de uma função de regressão linear.
    • Previsão de chuvas na Austrália: Esse projeto visa criar um modelo preditivo para analisar se haverá chuva amanha, usando uma base de dados da Austrália, criei alguns modelos e selecionei o mais efetivo para a tarefa.

      • Análise de dados:

    • Coca Cola: Criação de Dashboard em Power BI para analisar as vendas da multinacional Coca Cola nos Estados Unidos.
    • Prime vídeo:Criação de Dashboard gerencial em Power BI para analisar os filmes e series disponiveis na Prime Video.
    • COVID-19: Será que gênero e idade são fatores estatisticamente significativos para determinar a probabilidade de alguém morrer de COVID-19.
    • Comidas de Bengaluru: Zomato é um serviço de busca de restaurantes, nesse projeto irei explorar e visualizar dados dos restaurantes de Bengaluru na India.
    • Dashboard gerencial: BrightBuy é uma empresa de varejo renomada, conhecida pela sua variedade de produtos, qualidade e preços acessíveis. Foi solicitado pelo departamento comercial a construção de um Dashboard Gerencial para auxiliar na tomada de decisões do gerente de vendas. Feito com o RStudio e o Canva.

      • Outros:

    • Página web pessoal: Desenvolvimento de uma página web pessoal com informações sobre mim, interesses, disciplinas cursadas e projetos.

    • Dados faltantes Esse mini projeto visa mostrar como lidar com dados faltantes em bases de dados. Para isso selecionei três métodos de tratamento, o primeiro seria a remoção das linhas ou colunas com dados faltantes, o segundo preencher os dados faltantes com a média e o terceiro preencher com a mediana.
